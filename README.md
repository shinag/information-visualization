# Fast Fashion Environmental Impact Dashboard

An interactive data visualization dashboard analyzing the environmental footprint of major fast fashion brands across the globe. Built with D3.js and featuring 9 comprehensive visualizations covering carbon emissions, water usage, waste production, and sustainability metrics from 2015-2024.


## Overview

This dashboard provides a data-driven exploration of fast fashion's environmental impact, featuring:
- **10 Countries**: Manufacturing hubs across Asia, Europe, and Americas
- **5 Major Brands**: Shein, Forever 21, Uniqlo, Zara, H&M
- **10 Years**: Comprehensive data from 2015-2024
- **9 Interactive Visualizations**: From global maps to detailed trend analysis

## Key Features

### Interactive Visualizations

1. **Global Impact Map** - Choropleth visualization showing environmental impact index by country
   - Filter by impact level (High/Medium/Low)
   - Click-to-zoom functionality
   - Weighted impact calculation: 50% Carbon + 30% Water + 20% Waste

2. **Regional Comparison** - Bar chart comparing countries across metrics
   - Toggle between Carbon, Water, and Waste
   - Sort by highest/lowest values
   - Efficiency metrics per tonne of production

3. **Brand Rankings** - Top 15 brands by carbon emissions
   - Country-based filtering
   - Auto-play carousel through countries
   - Animated bar transitions

4. **Time Trends** - Dual-axis line chart tracking decade-long changes
   - Carbon emissions (solid line)
   - Water usage (dashed line)
   - Brand-specific filtering

5. **Sustainability Performance** - Multi-line chart showing brand progress
   - Based on actual Sustainability Scores (0-100)
   - Interactive brand selection
   - Hover details with environmental breakdowns

6. **Cost Efficiency Heatmap** - Brand volatility analysis
   - Color-coded efficiency patterns
   - Interactive row/column highlighting
   - Identifies stable vs. volatile performers

7. **Production vs Waste** - Scatter plot correlation analysis
   - Color-coded by brand
   - Waste-to-production ratios
   - Efficiency outlier identification

8. **Best Performers** - Parallel coordinates chart
   - 6-dimensional analysis
   - Brushable axes for filtering
   - Multi-metric sustainability comparison

9. **Economics vs Environment** - GDP contribution vs sustainability
   - Toggle between metrics
   - Trend line with linear regression
   - Quadrant analysis

## Getting Started

### Prerequisites

- Web browser (Chrome, Firefox, Safari, or Edge)
- No build tools required - runs directly in browser!
